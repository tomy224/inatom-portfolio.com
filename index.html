改善実績

  • パフォーマンス最適化: APIレート制限を考慮した並列処理設計

このポートフォリオサイト

×
2025年8月 - 現在

プロジェクト概要

AWS静的ウェブサイトホスティングのベストプラクティスを実装したポートフォリオサイト。 高可用性・低コスト・高パフォーマンスを実現する現代的なWeb構成を採用しています。

AWS構成

  • S3: 静的ウェブサイトホスティング、99.999999999%の耐久性
  • CloudFront: CDN、HTTPS終端、グローバル配信
  • Route53: DNS管理、100%のSLA保証
  • ACM: SSL証明書、無料で自動更新
  • GitHub Actions: CI/CD、自動デプロイ

パフォーマンス・コスト

  • 月額運用コスト: 約280円（ドメイン代込み）
  • ページ読み込み速度: 1秒未満を実現
  • 可用性: 99.99%以上
  • グローバル配信: 世界中のユーザーに高速配信

技術選択理由

なぜこの構成を選んだのか

  • S3: サーバー管理不要で高可用性、静的サイトには最適解
  • CloudFront: ユーザー体験向上とセキュリティ強化を両立
  • Route53: DNS障害によるサービス停止リスクを最小化
  • CI/CD: 手動デプロイのヒューマンエラー防止、開発効率向上

学んだこと

  • 静的サイトホスティングのベストプラクティス
  • CDNによるパフォーマンス最適化
  • CI/CDパイプラインの設計と運用
  • コスト最適化の重要性
